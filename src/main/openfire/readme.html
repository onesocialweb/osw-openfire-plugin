Onesocialweb Plugin Readme

Overview

Openfire plugin for the Onesocialweb protocol extendsion. Released under an Apache 2.0 License. Copyright 2010 Vodafone Group Services Limited.

Installation

Copy the plugin jar archive into the plugins directory of your Openfire installation. The plugin will then be automatically deployed. To upgrade to a new version, copy the new archive over the existing file.

Configuration

No configuration for now.
